# sistema-de-seguranca-residencial-utilizando-arduino
Sistema inteligente de segurança residencial com detecção de presença e fumaça utilizando Arduino Uno e arquitetura ARM.

https://wokwi.com/projects/464507831152079873

O projeto apresenta o desenvolvimento de um sistema automatizado de segurança residencial utilizando Arduino Uno e conceitos de arquitetura ARM. O objetivo do projeto foi criar um sistema capaz de monitorar ambientes residenciais por meio da detecção de presença e fumaça, emitindo alertas automáticos em situações de risco. Para isso, foram utilizados sensores como o PIR para detecção de movimento e o MQ-2 para identificação de fumaça e gases, além de componentes como LED, buzzer e display LCD para sinalização visual e sonora. O sistema foi desenvolvido utilizando linguagem C/C++ e simulado na plataforma Wokwi, permitindo validar o funcionamento do protótipo de forma prática. Também foi realizada uma simulação em arquitetura ARM por meio do CPUlator ARMv7, com o objetivo de compreender conceitos básicos relacionados ao processamento embarcado.
